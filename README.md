# Nim-Combinatorial-Game
Developed the Game of Nim, a mathematical combinatorial game using logic from combinatorial game theory with the Sprague-Grundy theorem. This Game of Nim features modular game architecture, robust input validation, dynamic game-state management, randomized board generation, and persistent score tracking across multiple rounds. The project demonstrates strong software engineering principles through clean class design, encapsulation, turn-based game logic, and interactive console-based user experiences. Developed with @kamillopez09.
